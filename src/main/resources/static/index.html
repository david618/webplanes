Webplanes

Returns simulated planes messages

Created as test inputs

  • /swagger-ui.html: Swagger UI
  • /planes : Get planes for this second. Usually returns 1,000 some planes with lat outside of -88 to 88 are not included.
  • /planes?num=50&var=5&rnd=true&gaussian=false : Return 50 random planes from the 1,000 plus or minus 5
